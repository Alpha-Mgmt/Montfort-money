# Montfort Money

Budget app — part of the Montfort family. Next.js + Tailwind + Supabase, deployed on Vercel at `money.montfortfinancial.com`.

**v1:** manual expense tracking, monthly budgets per category, and money tasks that book themselves into your budget when completed.

**Designed for what's next:** accounts/transactions carry `source` + `external_id` (Plaid-ready), per-user structured data with RLS (AI insights-ready).

See `GUIA-DANIEL.md` for setup & deploy.
